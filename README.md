## Snap.Nutrition

> This app was created using Claude Code by Anthropic, during a seminar-style event hosted by Rewriting the Code. This app is entirely AI-generated and has not been customized as of now. 

## Summary

This app is designed to simplify the act of tracking nutrition. There are so many apps for logging food intake whether that be for health reasons of for reaching gym goals. The issue is that most of those apps expect the user to log that information manually, which is time-consuming and can lead to individuals giving up on the task.

That's where Snap.Nutrition comes in. Just like it says, *snap* a photo, let the AI scan the image, and let estimate the nutrition information.

## How to run this
> The description on how to run this that is inside the project is very technical and I would like non-technical audiences to also be able to run this locally.
### Before you start
>> Make sure you have an Anthropic account (create for free at anthropic.com)
>> a personal access token from Anthropic (do not share this publicly!)

### Step by Step
> Step 1: Download the zip file from Github, unzip it, and open the folder inside

> Step 2: Find the file `demo.html` and open it in your browser (double-click it in the folder)

> Step 3: Get your personal access token
>> a - Go to console.anthropic.com, sign in, click your profile in the top right corner, select 'API Keys'
>> b - Click "Create Key," name it whatever you would like, and copy the key it gives you (it will look something like `sk-ant-...`). 
>> c - Remember: this is a a private key, treat it like a password

> Step 4: Paste the key into the field at the top of the demo page where it asks for the `Anthropic API Key`
>> A note on cost: Using the AI costs a very small amount per photo (a few cents at most). Anthropic gives new accounts some free credit to start with, so you likely won't need to pay anything to try the demo.

> Step 5: Upload a photo of any food and click `Analyse meal`

## Features
- A place to add your photo
- Nutrition information
- A percent-accuracy rating on how likely the AI can estimate the nutrition
